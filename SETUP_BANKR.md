# Bankr LLM Setup Guide

## 🚀 Setup Your Bankr LLM API Key

### Step 1: Add Your API Key
Replace `your_bankr_llm_api_key_here` in the `.env` file with your actual Bankr LLM API key:

```env
# OpenAI API Configuration (Bankr LLM)
OPENAI_API_KEY=your_actual_bankr_llm_api_key_here
OPENAI_BASE_URL=https://api.bankr.ai/v1
OPENAI_MODEL=gpt-5.4-mini
```

### Step 2: Restart the Server
```bash
# Stop the current server (Ctrl+C)
# Then restart:
node server.js
```

### Step 3: Test the Setup
1. Open your TweetIQ app: `file:///C:/Users/gaura/OneDrive/Desktop/claude/tweetiq-final.html`
2. Enter a Twitter username (e.g., `GUJJUIIXI`)

## 🔧 Troubleshooting

If you get API errors:
1. Check your Bankr LLM API key is correct
2. Ensure you have credits in your Bankr LLM account
3. The app will automatically fallback to basic analysis if AI fails
